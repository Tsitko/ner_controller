# Задача

Реализовать исследование уменьшения галлюцинаций LLM с использованием NER-модели.

## Клиент для LLM

Разработать клиент для LLM, который будет подключаться к LLM и отправлять запросы. Используем LLM на LM Studio 192.168.1.16:1234

Через клиент запросить модели и описать их в readme в этой папке.

## Исследование

Взять декларацию независимости США и попросить LLM дать краткое описание каждого пункта. Сохранить ответ и время обработки запроса.

Сделать запрос для контроля галлюцинаций LLM. Использовать http://localhost:1304/docs#/hallucination/handle_hallucination_check_hallucination_check_post

Первый ответ LLM, декларацию и ответ от сервиса контроля галлюцинаций повтоно отправить в ту же модель с запросом исправить галлюцинации и внести уточнения. Сохранить ответ и время обработки запроса.

Сделать это для:

qwen2.5-7b

gpt-oss-20b

gpt-oss-120b

qwe3-next

llama4-70b

## Результаты

Составить таблицу с результатами:
